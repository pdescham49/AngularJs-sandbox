AngularJs - User management demo application

Launch Demo - UserManagement

Using AngularJs 1.5 and Bootstrap Directives for angularJs I decided to build a fully functional UserManagement tool to showcase some of what angularJs has to offer; the functionality includes:

  • Edit / Create / Delete
  • In-line edit (quickedit)
  • Search
  • Sort / Filter
  • A CRUD interface
  • Random Test Data injection
