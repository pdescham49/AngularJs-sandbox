HiveUX - Grid and toolkit

Launch Demo - HiveUx

HiveUx - Is my own expermentation on creating a desktop envoirment in pure CSS / Javascript leveraging as much CSS-3 functionality as possible to minimize the use of javascript.

Features:

  • Responsive Grid - To address any mobile formfactor / orientation I devised a custom CSS framework that is percentage based.
  • Javascript Modules - The framework is module-based and each class self registers within the "core" class.; current modules include:
    • hive.js - Core lib
    • event.js - Event model
    • debug.js - Debug management
    • geo.js - GeoLocation lib
    • nav.js - Navigation
    • util.js - Utility class
    • ux.js - UX class - Window management and templating.
